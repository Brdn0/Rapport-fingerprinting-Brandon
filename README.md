# Rapport fingerprinting Brandon Nana
Fingerprinting et reconnaissance d'un serveur web, projet personnel

Ce rapport a été rédigé dans un objectif pédagogique et professionnel, afin de documenter un exercice de fingerprinting/reconnaissance d’un serveur web et d’en restituer les résultats de manière claire, structurée et reproductible.
Il s’appuie sur les consignes du TP (collecte d’informations, utilisation d’outils de reconnaissance, mise en évidence des éléments exposés et des contre-mesures) ainsi que sur une structure de rapport de type “penetration test report” (confidentialité, périmètre, approche, résultats, recommandations, annexes).
Les captures et sorties d’outils associées (annexes) servent de preuves et permettent de justifier les constats techniques présentés dans le document.
